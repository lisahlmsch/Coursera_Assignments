Introduction

I created a shiny application that allows the user to find out the GDP in million USD for a country of choice.

The application features an input widget to select the country of interest.

Data Source

The original data sources can be found here: http://data.worldbank.org/data-catalog/GDP-ranking-table

The download link is as follows: https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FGDP.csv
